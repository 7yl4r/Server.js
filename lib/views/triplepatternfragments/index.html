Welcome to DBpedia!

DBpedia is a crowd-sourced community effort to extract structured information from Wikipedia and make this information available on the Web. DBpedia allows you to ask sophisticated queries against Wikipedia.

Data from DBpedia can be accessed as four complementary kinds of Linked Data Fragments:

  • Data dumps, which you can download and query locally
  • SPARQL endpoint queries, which are executed by the DBpedia server
  • Linked Data documents, which you can browse by topic
  • Triple Pattern Fragments, which allow clients to execute queries

This page is the entry point of the Triple Pattern Fragments interface.

Browse DBpedia by triple pattern

Use the DBpedia triple pattern fragments interface to query by triple pattern. Examples:

  • Show all artists
  • Show all things that link to Belgium
  • List all people's birth places

Query DBpedia from your browser

Your browser executes these queries locally using triple pattern fragments.
